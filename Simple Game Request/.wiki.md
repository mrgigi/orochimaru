# Project Summary
The project is an interactive battle game titled **Orochimaru: Serpent's Wrath**, centered around the character Orochimaru from the Naruto lore. It aims to engage players in combat scenarios supporting the upcoming launch of the Orochimaru token. The game features various jutsu attacks, enemy waves, and a health and chakra system, all designed to provide an immersive gaming experience while promoting the token.

# Project Module Description
The project consists of several functional modules:
- **Game Engine**: Manages game state, player interactions, and rendering.
- **Player**: Handles player state, movement, and attack mechanics.
- **Enemy**: Manages enemy creation, behavior, and rendering.
- **Types**: Defines various interfaces for game entities, attacks, and states.

# Usage
To set up the project, follow these steps:
1. Install dependencies using the package manager:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Start the development server:
   ```bash
   pnpm run dev
   ```
