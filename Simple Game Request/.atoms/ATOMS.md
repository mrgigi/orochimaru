# Project Context

## Project Overview
Orochimaru: Serpent's Wrath - An interactive side-scrolling battle game where players control Orochimaru fighting waves of ninja enemies. Built to support an upcoming Orochimaru token launch. Features multiple attack types, wave-based enemies, health/chakra system, and token branding.

## Constraints
- Color Palette: Dark purple (#1a0a2e, #2d1b4e), toxic green (#39ff14, #00ff41), deep black (#0a0a0a), gold accents (#ffd700)
- Typography: Bold, angular fonts for headings; clean sans-serif for UI elements
- Visual Elements: Snake-scale patterns, glowing neon effects, particle systems, dark atmospheric backgrounds
- Layout: Full-screen game canvas with HUD overlay (health/chakra bars top, score top-right, controls bottom on mobile)
- Max 8 code files
